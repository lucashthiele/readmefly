> ReadmeFly is a Software as a Service that provides software documentation powered by AI.

## Prerequisites

- node >=21.7.3
- npm >=10.5.0
- yarn >=1.22.22

## Install

```sh
yarn
```

## Usage

```sh
yarn start
```

## Run tests

```sh
yarn test
```
